Dockmodal - Gmail like Dockable Modal Dialog Plugin

Modal dialogs that can be minimized

  • Created: 01/01/2014
  • Latest update: 02/10/2014
  • By: Tarafder Ashek E Elahi
  • Email: mail.ashek@gmail.com

Thank you

Basic Usage

Step 1

Add the jquery.dockmodal.css in your document head

<link href="assets/css/jquery.dockmodal.css" rel="stylesheet"/>

Step 2

Add the jquery.dockmodal.js after loading jquery.js

<script src="assets/js/jquery.dockmodal.js"></script>

Step 3

Then initialize the docked modal. It's pretty simple. Just pick any html element and call .dockmodal() on its jQuery instance within your javascript code

$("#dialog1").dockmodal();

There are lots of options that can be passed to the initializer. For a detailed list scroll down to options section.

$("#dialog1").dockmodal({
    title: "Window 1",
    buttons: [
        {
            html: "Cancel",
            buttonClass: "btn btn-default",
            click: function () {
                // do something when the button is clicked
            }
        }, {
            html: "Ok",
            buttonClass: "btn btn-default",
            click: function () {
                // do something when the button is clicked
            }
        }
    ]
});

Examples

Basic usage with default options

Call dockmodal on the element you want to show. By default it will be visible as a dialog. You can dock, minimize or close the dialog.

$("#dialog1").dockmodal();

Specify initial state

Just pass an extra parameter stating the initial state of the modal. Possible states are "modal", "docked" and "minimized". You will find more on this param at Options section

$("#dialog2").dockmodal({
    initialState: "docked"
});
$("#dialog3").dockmodal({
    initialState: "minimized"
});

Specify dimension

Specify width and height at initialization. You will find more on this param at Options section

$("#dialog4").dockmodal({
    initialState: "docked",
    width: 400,
    minimizedWidth: 200,
    height: "40%",
    title: "Custom dimension 1"
});
$("#dialog5").dockmodal({
    initialState: "docked",
    width: 500,
    minimizedWidth: 300,
    height: "50%",
    title: "Custom dimension 2"
});

Customize title bar icons

Specify which action icons you want to be shown at title bar. You will find more on this param at Options section

$("#dialog6").dockmodal({
    initialState: "modal",
    title: "Modal - close only",
    showClose: true,
    showPopout: false,
    showMinimize: false
});
$("#dialog7").dockmodal({
    initialState: "docked",
    title: "Docked - close only",
    showClose: true,
    showPopout: false,
    showMinimize: false
});

Action buttons at footer

You can use action buttons at footer and bind events to it. You will find more on this param at Options section

$("#dialog8").dockmodal({
    initialState: "docked",
    title: "Dialog with footer actions",
    width: 400,
    buttons: [
        {
            html: "Cancel",
            buttonClass: "btn btn-primary",
            click: function (e, dialog) {
                // do something when the button is clicked
                dialog.dockmodal("close");
            }
        }, {
            html: "Ok",
            buttonClass: "btn btn-primary",
            click: function (e, dialog) {
                // do something when the button is clicked
                dialog.dockmodal("close");
            }
        }
    ]
});

Content loading via AJAX

You can load html data via ajax and bind dockmodal on it. Find the following example. For simplicity lets call jQuery's .get() method and load content via ajax. At load callback, create a blank div and push the returned html within it. Finally call dockmodal() on this instance. You will find more on this param at Options section

$.get("ajax-content.txt", function(data){
    $("<div/>").html(data).dockmodal({
        initialState: "docked",
        title: "Ajax content",
        width: 300,
        buttons: [
            {
                html: "Cancel",
                buttonClass: "btn btn-primary",
                click: function (e, dialog) {
                    // do something when the button is clicked
                    dialog.dockmodal("close");
                }
            }, {
                html: "Ok",
                buttonClass: "btn btn-primary",
                click: function (e, dialog) {
                    // do something when the button is clicked
                    dialog.dockmodal("close");
                }
            }
        ]
    });
});

Events callback

You can hook to a bunch of events which get fired from times to time at different actions. The following example shows open, minimize and close events. For a complete list please refer to the Events section

$.get("ajax-content.txt", function(data){
    $("<div/>").html(data).dockmodal({
        initialState: "docked",
        title: "Ajax content",
        width: 300,
        buttons: [
            {
                html: "Cancel",
                buttonClass: "btn btn-primary",
                click: function (e, dialog) {
                    // do something when the button is clicked
                    dialog.dockmodal("close");
                }
            }, {
                html: "Ok",
                buttonClass: "btn btn-primary",
                click: function (e, dialog) {
                    // do something when the button is clicked
                    dialog.dockmodal("close");
                }
            }
        ]
    });
});

Options

Options are passed during the initialization of the dialog

Code examples:

Initialize the dialog with the options specified

$("#dialog1").dockmodal({
    width: 400,
    height: "65%",
    minimizedWidth: 200,
    gutter: 10,
    poppedOutDistance: "6%",
    title: "My own dialog",
    dialogClass: "custom-dialog",
    animationSpeed: 400,
    initialState: "modal",
    showClose: true,
    showPopout: true,
    showMinimize: true,
    buttons: [
        {
            html: "Cancel",
            buttonClass: "btn btn-default",
            click: function () {
                // do something when the button is clicked
            }
        }, {
            html: "Ok",
            buttonClass: "btn btn-default",
            click: function () {
                // do something when the button is clicked
            }
        }
    ]
});

Option Name Value Type Default Value Comment
width Number 400 Pixel amount
height String/Number 65% Percentage or pixel amount.
minimizedWidth Number 200 Pixel amount
gutter Number 10 Pixel amount
poppedOutDistance String 6% Percentage or pixel amount. The distance from viewport to modal window.
title String "" Dialog title. If not specified, the title attribute of the modal element will be used.
dialogClass String "" Custom dialog class name
buttons Array [] Array of Button objects to be generated at footer. If passed empty then footer will not be generated at all. Each button object expects the following params
  • id: Button ID
  • html: Button text
  • buttonClass: Button class
  • click: Callback function with params (e, dialog)
animationSpeed Number 400 Number in miliseconds. Use 0 if you want no animation.
initialState String "modal" Initial state of the modal. Possible values: "modal", "docked", "minimized"
showClose Boolean true Use false to hide the close button
showPopout Boolean True Use false to hide the popout button
showMinimize Boolean True Use false to hide the minimize button

 

Methods

close()

Removes the dialog functionality completely. This will return the element back to its pre-init state.

Code examples:

Invoke the close method:

$("#dialog1").dockmodal("close");

minimize()

Minimizes the dialog.

Code examples:

Invoke the minimize method:

$("#dialog1").dockmodal("minimize");

restore()

Restores the dialog to its normal opened state.

Code examples:

Invoke the restore method:

$("#dialog1").dockmodal("restore");

popout()

Pops out the dialog to cover full screen and on to of other page components.

Code examples:

Invoke the popout method:

$("#dialog1").dockmodal("popout");

refreshLayout()

Refreshes the element position and recalculates.

Code examples:

Invoke the refreshLayout method:

$("#dialog1").dockmodal("refreshLayout");

Events

By default all callbacks receive the event and the dialog element as parameter.

Code examples:

Initialize the dialog with the callback specified. All the events work the same way.

$("#dialog1").dockmodal({
    create: function( event, dialog ) {}
});
Event Name Event params Callback details
create   Triggered when dialog is created
open   Triggered when dialog is opened
beforeClose   Triggered before closing the dialog. Return false from this function to prevent the dialog from closing.
close   Triggered after closing the dialog.
beforeMinimize   Triggered before minimizing the dialog. Return false from this function to prevent the dialog from minimizing.
minimize   Triggered after minimize action is done.
beforeRestore   Triggered before restoring the dialog. Return false from this function to prevent the dialog from restoring.
restore   Triggered after restore action is done.
beforePopout   Triggered before popping out the dialog. Return false from this function to prevent the dialog from popping out.
popout   Triggered after popout action is done.
